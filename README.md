## Viewing Prototypes

**On your phone (recommended):**
Open the GitHub Pages URL in mobile Safari/Chrome, then "Add to Home Screen" for a full-screen app experience.

🔗 **Live:** https://keithbfleming.github.io/doomscroll/prototypes/instagram-clone/

**Locally:**
```bash
# Any simple HTTP server works
cd prototypes/instagram-clone
python3 -m http.server 8000
# Open http://localhost:8000 on your phone (same Wi-Fi)
```
